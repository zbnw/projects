        HOW IT WORKS / 01
        
公开算法，保密密钥

        
Time Encrypt 不依赖隐藏算法。共享口令先派生主密钥，再结合 Unix 分钟编号生成当前分钟的 AES-256-GCM 密钥。签发时间与明文一起被认证加密，因此攻击者不能在不破坏认证标签的情况下修改时间。


        
    
          
  1. 派生PBKDF2 提高猜口令成本，HKDF 将主密钥隔离成每分钟独立密钥。
    2. 
          
  2. 封装签发时间、明文和随机填充组成固定 48 字节数据帧。
    3. 
          
  3. 加密AES-256-GCM 同时提供机密性与完整性，每条消息使用随机 IV。
    4. 
          
  4. 过期解密成功后检查受保护的签发时间，达到 900 秒立即拒绝。
